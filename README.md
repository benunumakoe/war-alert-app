# War Alert Tracker App

Real-time war alert and notification system for the Middle East region.

## Features
- Real-time push notifications for alerts
- Interactive map showing affected areas
- Location-based proximity alerts
- Filter alerts by type (rocket, drone, ground)
- Offline support
- Multiple language support (English, Hebrew, Arabic)

## Installation

### Prerequisites
- Node.js 16+
- React Native CLI
- Android Studio (for Android development)
- Xcode (for iOS development, Mac only)
